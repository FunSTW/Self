專案開發

2021年12月 ~ 2022年05月

Oculus Quest2 VR ︱ 物質密室逃生記 ︱ 國小 VR 教案

遊戲獨立開發(美術設計除外)，使用Unity引擎，使用硬體為Oculus Quest2，平台為Android。

由於目標平台是使用Android開發VR遊戲，必須注意GPU的Based-Tiles優化，如避免使用延遲渲染、預深度通道(破壞Early-Z)、避免後期處理、幾何著色器(Geometry shader)。

2018年06月 - 2020年06月

Vive VR ︱ BehindTheSoundVR ︱ 大學專題開發

利用Unity開發的VR密室逃脫恐怖遊戲，我在團隊裡擔當程式設計、撰寫 Shader、製作特效、編輯企劃書、建模等角色。

在開發的這段期間採用與學習的許多技術。例如：
使用者介面 ( UI ) 與使用者經驗 ( UX ) 呈現方法、視野與介面FOV與距離、頭戴顯示器穿牆處理...。
細節層次( LOD )、遮擋剃除 ( Occlusion culling )、光照貼圖烘焙 ( Lightmapping )、亮度探針 ( Light probe group)、反射參考 ( Reflection probe )、3D圖學、Shader撰寫；與其他插件的使用：Shader Graph、Post Processing、ProBuilder、Timeline、Cinemachine、Universal Render Pipeline...。
基本 PBR 材質繪製、高模烘焙低模。
