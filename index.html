專案開發 X 作品集

2018年06月 - 2020年06月

大學專題 BehindTheSoundVR 開發

　　利用Unity開發的VR密室逃脫恐怖遊戲，我在團隊裡擔當程式設計、編輯企劃書、建模等角色。在開發的這段期間採用與學習的許多技術。例如：

VR：
　　使用者介面 ( UI ) 與使用者經驗 ( UX ) 呈現方法、視野與介面FOV與距離、頭戴顯示器穿牆處理...。
Unity：
　　細節層次( LOD )、遮擋剃除 ( Occlusion culling )、光照貼圖烘焙 ( Lightmapping )、亮度探針 ( Light probe group )、反射參考 ( Reflection probe )、3D圖學、Shader撰寫；與其他插件的使用：Shader Graph、Post Processing、ProBuilder、Timeline、Cinemachine、Universal Render Pipeline...。
建模：
　　基本 PBR 材質繪製、高模烘焙低模。
